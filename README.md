# jQuery Plugin: Confirmable Clicks

Add a confirmation dialog to any clickable element

## Example Usage

$('.clickable').confirmable();

## Options

* title
* action

    $('.clickable').confirmable({  
        title: 'Are you sure you want to delete this item?',  
        action: 'Yes'  
    });
